The EDI-FICATION working principle

Our software uses "Computer Vision" to extract text from pdf or images submitted by users using python API(pypdf2) and then eliminate common words like "the","be","what" using dictionary(pre stored datastructure) and sort remaining words using "nltk" and "CountVectorizer" then uses flask(python's library) to give user important topic as output

Book exchange

We have connected our software to firebase database, and store data of books in json format, every time a new book is added, a new node is created storing data of all users who want to give that book
